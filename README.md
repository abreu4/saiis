# AIAS Website

This is the official website for AIAS, built with Hugo.

## Prerequisites

- Hugo (extended version recommended)
- Git

## Installation

### Install Hugo

```bash
# Install Hugo via Homebrew (macOS)
brew install hugo

# Verify installation
hugo version
```

## Development

### Run the Development Server

```bash
# Start the Hugo development server
hugo server

# The site will be available at http://localhost:1313
```
